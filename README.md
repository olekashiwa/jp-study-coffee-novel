## 📚 Описание проекта

Это приложение создано для глубокого изучения японского языка через аутентичный литературный текст. Роман «コーヒーが冷めないうちに» содержит богатый языковой материал уровней N3-N1: живые диалоги, эмоциональные монологи, описания и философские размышления.

Приложение предлагает комплексный подход к изучению:
- 📖 **Чтение** оригинального текста с параллельным переводом
- 📚 **Лексика** с примерами и контекстом
- 🔤 **Грамматика** с подробными объяснениями
- 🈳 **Кандзи** с чтениями и значениями
- ✍️ **Упражнения** для проверки понимания
- 🃏 **Флэшкарты** с интервальным повторением

## ✨ Особенности

- **Структурированные данные** — лексика, грамматика, кандзи по главам
- **Система тестирования** — проверка знаний учащихся

| Уровень | Описание | Материалы |

| **N1** | Экспертный уровень | Редкая лексика, сложные грамматические конструкции, литературный язык |

## 💻 Установка

### Системные требования
- Python 3.8 или выше
- Оперативная память: 256 МБ (рекомендуется 512 МБ)
- Место на диске: 100 МБ

### Пошаговая установка

**1. Клонируйте репозиторий**
```bash
git clone https://github.com/yourusername/japanese-learning-app.git
cd japanese-learning-app
```

**2. Создайте виртуальное окружение (рекомендуется)**
```bash
# Windows
python -m venv venv
venv\Scripts\activate

# Linux/Mac
python3 -m venv venv
source venv/bin/activate
```

**3. Установите зависимости**
```bash
# Основные зависимости (встроены в Python)
# Ничего устанавливать не нужно, всё уже есть!

# Дополнительные зависимости (опционально)
pip install -r requirements.txt
```

**4. Запустите приложение**
```bash
python main.py
```

## 🧠 Система интервального повторения

Приложение использует модифицированный алгоритм **SM-2** (SuperMemo-2) для эффективного запоминания лексики.

### Как это работает

1. Новая карточка появляется в очереди на сегодня
2. Вы видите слово и пытаетесь вспомнить значение
3. Оцениваете сложность:
   - **😊 簡単** (легко) — интервал увеличивается
   - **😐 普通** (нормально) — стандартный интервал
   - **😓 難しい** (трудно) — карточка возвращается в короткую очередь

### Интервалы повторения

| Уровень | Интервал |
|---------|----------|
| Новая карточка | Сегодня |
| После 1-го повторения | 1 день |
| После 2-го повторения | 3 дня |
| После 3-го повторения | 7 дней |
| После 4-го повторения | 14 дней |
| После 5-го повторения | 30 дней |
| После 6-го повторения | 90 дней |
| Мастер | 180+ дней |

## 🔧 Технические детали

### Технологический стек
- **Язык**: Python 3.8+
- **GUI**: Tkinter
- **База данных**: SQLite3
- **Архитектура**: MVC (Model-View-Controller)

### Основные компоненты
- **DatabaseManager** — управление подключением к БД
- **DataLoader** — загрузка данных из JSON
- **FlashcardService** — алгоритм SM-2
- **ExerciseService** — генерация упражнений
- **ProgressService** — отслеживание прогресса

### Форматы данных
- **JSON** — хранение лексики, грамматики, упражнений
- **SQLite** — пользовательские данные, прогресс, карточки
